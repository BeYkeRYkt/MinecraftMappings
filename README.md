MinecraftMappings
=================
Automatically generated minecraft mappings, using [SrgLib](https://github.com/ProjectTestificate/SrgLib)

## Commands used to generate:
- 1.8: python3 genmappings.py 1.8 snapshot_nodoc_20151122
